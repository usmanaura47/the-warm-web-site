## Running locally

You need Ruby and gem before starting, then:

```bash
# install bundler
gem install bundler

# clone the project
git clone https://github.com/usmanaura47/the-warm-web-site.git
cd the-warm-web-site

# run jekyll with dependencies
bundle exec jekyll serve
```
